# 我的閱讀 MOC

書當 wikilink 互相連，不用為了寫筆記去湊資料。

## 📚 在讀
```dataview
TABLE author, started
FROM "20-Areas/我的閱讀"
WHERE type = "book" AND status = "reading"
SORT started DESC
```

## ✅ 已讀完
```dataview
TABLE author, rating, finished
FROM "20-Areas/我的閱讀"
WHERE type = "book" AND status = "finished"
SORT finished DESC
LIMIT 20
```

## 💤 暫停 / 棄
```dataview
TABLE author, started
FROM "20-Areas/我的閱讀"
WHERE type = "book" AND status = "paused" OR status = "dropped"
```

## ⭐ 高評分（4+ 星）
```dataview
TABLE author, rating, finished
FROM "20-Areas/我的閱讀"
WHERE type = "book" AND rating >= 4
SORT rating DESC, finished DESC
```

## 📊 統計
```dataview
TABLE WITHOUT ID 
  length(rows) AS "本",
  rows.status[0] AS "狀態"
FROM "20-Areas/我的閱讀"
WHERE type = "book"
GROUP BY status
```

## 怎麼新增一本書
1. `Ctrl+P` → `Templater: Create new note from template` → `Book-template`
2. 把檔案名改成書名（按 `F2` 重命名）
3. 填 frontmatter：`author`, `rating`（1–5）, `status`（reading/finished/paused/dropped）
4. 寫筆記（三個觸動點子那段最重要）
5. 存檔 — 自動出現在上面表格
