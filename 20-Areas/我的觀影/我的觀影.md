# 我的觀影 MOC

## 🎬 最近看的
```dataview
TABLE director, year, rating, watched
FROM "20-Areas/我的觀影"
WHERE type = "movie"
SORT watched DESC
LIMIT 20
```

## ⭐ 高評分（4+ 星）
```dataview
TABLE director, year, rating
FROM "20-Areas/我的觀影"
WHERE type = "movie" AND rating >= 4
SORT rating DESC
```

## 📊 按年份分
```dataview
TABLE WITHOUT ID
  year AS "年份",
  length(rows) AS "看過"
FROM "20-Areas/我的觀影"
WHERE type = "movie"
GROUP BY year
SORT year DESC
```
