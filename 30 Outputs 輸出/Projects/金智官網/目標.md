## 3. 品牌目標

### 對外定位
- **主品牌**：智廷工程行（公司登記名稱、信任感）
- **副品牌**：金智消防（口碑、SEO 承接）
- **slogan**：北北基專業消防工程・檢修申報
- **差異點**：負責人郭金智 20+ 年實戰、業主與管委會兩種客群通用、一條龍服務

### 不要做的事（紅線）
- 🚫 不寫「消防設備師證照加持」（客戶定位是消防工程廠商，不是設備師）
- 🚫 不把「公司 20+ 年」掛嘴邊（公司 2023 設立、個人才 20+ 年；公平交易法切開）
- 🚫 不放 Google Map（住商混合地址）

## 4. 客戶目標（誰來找我們）

### 主力客群 A：業主 / 店家
- 痛點：建物公安申報壓力、消防改善被罰款、想開店過勘驗
- 進站關鍵字：基隆消防 / 店面消防 / 公安申報
- CTA 路徑：首頁業主 Tab → 服務頁 → 詢問表單

### 主力客群 B：社區管委會 / 主委
- 痛點：消防設備老舊待汰換、年度檢修申報、住戶投訴
- 進站關鍵字：社區消防 / 管委會 消防改善 / 公寓消防安檢
- CTA 路徑：首頁管委會 Tab → 服務頁 → 詢問表單
