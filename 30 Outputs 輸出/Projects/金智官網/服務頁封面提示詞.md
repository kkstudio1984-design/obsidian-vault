# 金智官網｜服務頁封面圖提示詞

> 6 篇服務頁各一張封面圖（1200×630），對應已上線的 `/services/<slug>` 詳情頁。  
> 風格基準同站上其他底圖：黑金 luxury（`#0f0f0f` 黑底 + `#c9a876` 金）、無人臉、紀錄片質感。  
> 工具不挑：MJ / Firefly / DALL-E 3 / Flux 都可。下方 prompt 已照 MJ 語法寫，其他工具去掉 `--ar` 與 `--style raw` 即可。

## 設計原則（豐富畫面但不雜亂）

每張採三層構圖：

1. **前景 hero**：1 個主物件，與服務主題直接呼應
2. **中景配件**：3–5 個敘事道具，建立「正在發生這件事」的場景感
3. **背景環境**：氛圍（地點、時間、材質），用景深虛化但有可辨識輪廓
4. **光線氛圍**：暖金重點光、深黑陰影做對比
5. **限制**：no people / no faces / no readable text / no brand logos

## 產完之後怎麼上線

1. 圖檔導出 PNG / JPG（1200×630 或 16:9）
2. 進 Notion「SEO 文章管理」DB → 開對應 slug 的服務頁
3. 「封面圖」欄位上傳該圖
4. 按頁面右上「🚀 立刻發布」按鈕
5. ~10 秒後 `https://kingzhi-web.vercel.app/services/<slug>` 自動帶入

> Notion 圖片在 build 時會自動下載到 `public/images/notion/<hash>.<ext>` 本地化（避免 S3 簽名 1 小時過期）。
