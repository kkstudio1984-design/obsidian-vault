## 4. 核心頁面清單（按優先順序）

### ⭐ P1 — 首頁 `/`

由上到下區塊：
1. **頂部 Navbar**（深藍底白字）：Logo 文字「金智消防」+ 主選單（服務 / 知識庫 / 案例 / 聯絡）+ 手機版 hamburger
2. **Hero 區**：
   - H1「北北基專業消防工程、檢修申報一條龍服務」
   - Sub「負責人郭金智 20+ 年實戰經驗，親赴現場」
   - 雙 CTA 按鈕：`📞 立即撥打 0925-369-737`（橘底）+ `💬 LINE 詢問`（白底綠字）
   - 背景建議：實景照（消防設備檢查現場）加暗色疊加，或純色 + 幾何裝飾
3. **三大服務卡軸**（3 欄 grid，手機單欄）：
   - 🧯 消防安全設備檢修申報
   - 🏗️ 廠房／商辦消防工程
   - 🏠 社區集合住宅改善
   - 每卡：icon + 標題 + 2 行介紹 + 「了解更多」連結
4. **信任訊號區**（橫式條狀）：`20+ 年實戰` / `基隆在地` / `24 小時回覆` / `北北基服務`
5. **最新知識 6 篇**（3×2 grid）：每篇卡片含封面圖、標題、上稿日、3 行摘要
6. **底部 CTA 橫幅**：再一次推「立即撥打」
7. **Footer**（深藍底）：公司資訊（見 §5）

### ⭐ P1 — 知識文章詳情 `/knowledge/[slug]`

這頁是 SEO 金雞母，要**特別精緻**。由上到下：
1. 麵包屑（首頁 → 知識庫 → 文章標題）
2. H1 大標
3. Meta 列：📅 上稿日 · 🔄 更新日 · 👤 郭金智 · ⏱ 閱讀時間
4. 封面圖（16:9）
5. **「⬆ 回到支柱文章：XXX」提示條**（只在群集文章出現，淡灰底）
6. **文章內文**（max-w-3xl，prose 樣式）：支援 H2/H3/引用/清單/圖片/code
7. **作者簡介卡**（深灰底 + 橘色左邊框）：郭金智大頭照（占位）+ 職稱 + 20+ 年經歷 + 代表案例
8. **延伸閱讀 3 篇**（3 欄卡片）
9. **側邊黏著 CTA**（桌機右側浮動，手機版變底部浮動列）
   - 業主版：「立即預約現場勘查」
   - 管委會版：「立即索取申報報價單」

### ⭐ P1 — 聯絡我們 `/contact`

1. H1 + 簡短引言
2. **聯絡表單**（左）：姓名 / 手機（必填）/ Email / 目標客群（radio：業主 or 管委會）/ 需求（textarea）
3. **聯絡資訊卡**（右）：
   - 📞 王小姐 0925-369-737（可一鍵撥打）
   - 💬 LINE 官方帳號 QR code
   - ✉️ a0925369737@gmail.com
   - 📍 基隆市安樂區（僅顯示區域，**不放 Google Map**）
4. 送出按鈕要大、橘色、佔滿表單寬度

### P2 — 知識庫總覽 `/knowledge`

四大 Topic Cluster 入口，每區塊顯示 6 篇卡：
- 🔴 商業搜尋詞
- 🟡 痛點／流程詞
- 🟠 檢修申報詞
- 🔵 地區詞

每個 cluster 用不同小點綴色，整體仍統一在深藍+橘主色系內。

### P2 — 服務總覽 `/services` / 案例總覽 `/cases`

卡片列表，結構類似 `/knowledge` 但不分 cluster。

### P3 — 404 頁

友善訊息 +「熱門文章 6 篇」+ 回首頁 CTA。
