# 金智官網｜知識文章封面圖提示詞

> 對應 /knowledge/<slug> 的 8 篇 SEO 文章，每篇 1200×630（1.91:1）封面。  
> 風格沿用案例頁 v2「動感版」三層構圖、黑金 luxury、no people / no faces / no readable text。  
> 知識文章封面要**比案例更抽象、更主題化**（不必有特定工地場景），方便讀者 Google 搜尋時識別主題。

## 設計原則

| 元素 | 知識文章 vs 案例頁差別 |
| --- | --- |
| 場景具體性 | 知識文章可較抽象（圖像化概念），案例頁要有現場感 |
| 焦點數 | 1 個 hero + 1 個輔助（不像案例頁有複雜道具） |
| 主題暗示 | 用標誌性物件代表主題（檢查表 = 申報、印章 = 送審、地圖 = 地區） |
| 動感 | 仍要 mid-action（翻頁、光柱、文件展開），不要靜止 |

## 產完後流程

1. 圖檔導出 PNG / JPG（1200×630 或 16:9）
2. 改成檔名 `<slug>.png`
3. 丟給我 → 我 cp 到 `public/images/knowledge/<slug>.png`
4. **方案 A**：直接 push → 自動 fallback 到本地圖
5. **方案 B**：在 Notion 該文「封面圖」欄位上傳 → override 本地圖
