# 金智官網｜案例頁封面圖提示詞 v2（動感版）

> 6 篇案例頁各一張封面圖（1200×630），對應已上線的 `/cases/<slug>` 詳情頁。  
> v2 強化「動感」：mid-action 瞬間、戲劇光感、新舊對比、多焦點構圖、間接人物臨場（不出現人臉但有「正在發生」的痕跡）。  
> 風格基準仍是黑金 luxury（`#0f0f0f` + `#c9a876`）、no people / no faces / no readable text。

## v2 設計原則

| 元素 | 怎麼做 |
| --- | --- |
| **動感** | 抓「mid-action 瞬間」：壓力錶飆升、雷射光束穿過粉塵、燈條測試亮起、煙霧浮動、水滴落下 |
| **戲劇光感** | 暖金重點光 + 冷藍/綠副光（會議室、機房）形成色溫對比；窗外光柱、燈罩光暈、scrim 光 |
| **新舊對比** | 同畫面內並陳：舊銹蝕 vs 新亮面、毛胚 vs 精緻、被破壞 vs 修復 |
| **多焦點** | 前景 hero + 中景「故事第二焦點」（半空咖啡杯、剛掛起的工地外套），讓視線多停一秒 |
| **間接人物** | 工具半握、椅子拉開、紙張被風翻起、影子斜過 — 顯示「人剛離開或正在動作中」 |
| **景深堆疊** | 前清晰、中軟焦、遠虛化金色 bokeh，三層空間感 |
| **粉塵 / 水霧** | 加入懸浮粒子、輕煙、噴霧線，避免乾淨無感的合成感 |

## v2 vs v1 差別速查

| 維度 | v1 | v2 |
| --- | --- | --- |
| 主物件 | 靜態安裝完成的設備 | mid-action（測試中、亮燈中、半安裝中） |
| 視覺焦點數 | 1 個 | 2 個（hero + 故事第二焦點） |

| 大氣感 | 乾淨 | 粉塵、水霧、蒸氣、光束粒子 |
