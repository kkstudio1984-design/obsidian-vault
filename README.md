# USER Obsidian Vault

個人筆記中心。PARA 系統改編。

## 資料夾

| 路徑 | 用途 |
| --- | --- |
| `00-Inbox/` | 收件匣，臨時想法先丟這 |
| `10-Projects/` | 各專案筆記（有結束日的事） |
| `20-Areas/` | 持續關注的領域（工作、健康、投資...） |
| `30-Resources/` | 參考資料、書摘、文章 |
| `40-Archives/` | 已完成或不再活躍的 |
| `90-Templates/` | 範本（Templater plugin 用） |
| `99-Daily/` | 每日日記 |

## 同步

GitHub private repo + Obsidian Git plugin。每 30 分鐘自動 commit + push。

## 跨裝置

- **電腦（新機）**：`git clone` 此 repo → Obsidian Open folder as vault
- **手機**：Obsidian Sync（$8/mo）或 Mobile Git plugin
